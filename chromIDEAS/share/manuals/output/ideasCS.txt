1. chromIDEAS.emission.txt: The emission parameters for chromatin states HMM model.
  - col1: Chromatin State label.
  - col2: Percentage of each chromatin state on the genome.
  - col3 - end: Probability of occurrence for each marker.

2. chromIDEAS.emission.pdf: Visualization of emission parameters for chromatin states HMM model.

3. chromIDEAS.state: Chromatin states and position classes. Separated by blank space.
  - col1: Bin ID
  - col2: Chromosome name
  - col3: Start (0-based)
  - col4: End (0-based)
  - col5 - col(5+n): Chromatin states label, where n=total number of cell types.
  - col last: The position class label.
  All labels starts from 0 (instead of 1).

4. .para (Hidden Folder): The temporary files of the calculation process.
  1) chromIDEAS.para0 & chromIDEAS.profile0: Merged parameters info from 100 random pre-runs.
  2) chromIDEAS.tmp.{1..100}.para: Parameters info from 100 random pre-runs.
  3) chromIDEAS.para & chromIDEAS.profile: Raw parameters info during during calculation.
  4) chromIDEAS.cluster: Local cell type clustering result, one row for each replicate of cell type.
  5) err.log & log.txt & std.log: The log files during calculation.
